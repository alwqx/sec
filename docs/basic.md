# 证券搜索与信息查询

## sec search (`sec s`)

搜索证券代码和名称，数据来源于[新浪财经](https://finance.sina.com.cn)。

```
sec search <keyword>
sec s <keyword>
```

### 参数

| 参数 | 说明 |
|------|------|
| `<keyword>` | 搜索关键词，支持代码（`600036`）、名称（`招商银行`）、拼音（`zsyh`） |

### 选项

| 选项 | 说明 |
|------|------|
| `-D, --debug` | 开启 debug 日志 |

### 实现

```
cmd/cmd.go (SearchHandler)
provider/sina/sina.go (Search)
```

调用 `sina.Search(ctx, key)` 搜索新浪财经接口，解析返回的证券列表。

### 输出字段

| 列名 | 说明 |
|------|------|
| 证券代码 | 带交易所前缀的代码，如 `SH600036`、`HK09988` |
| 证券名称 | 证券中文名称 |
| 证券类型 | `stock`（股票）/ `fund`（基金） |
| 交易所 | 交易所代码：sh/sz/bj/hk/ny/nasdaq |

### 输出示例

```bash
$ sec s 招商银行
证券代码    | 证券名称 | 证券类型 | 交易所
SH600036   | 招商银行 | stock   | sh
```

## sec info (`sec i`)

查询证券基本信息，数据来源于[新浪财经](https://finance.sina.com.cn)。

```
sec info <code>
sec i <code>
```

### 参数

| 参数 | 说明 |
|------|------|
| `<code>` | 证券代码，如 `600036`、`sh600036`、`09988` |

### 选项

| 选项 | 说明 |
|------|------|
| `-d, --dividends` | 同时显示分红送配历史 |
| `-D, --debug` | 开启 debug 日志 |

### 实现

```
cmd/cmd.go (InfoHandler)
provider/sina/sina.go (Search, Profile, QueryDividends)
```

1. `sina.Search(ctx, key)` — 搜索证券代码，取第一个结果
2. `sina.Profile(ctx, opts)` — 查询公司概况（合并 quote + profile + corp info）
3. 若 `-d` 指定：`sina.QueryDividends(ctx, code)` — 查询分红历史

### 输出字段

基础信息（`fmt.Printf` 格式化输出）：

| 字段 | 说明 |
|------|------|
| 证券代码 | 带交易所前缀的代码 |
| 简称历史 | 历史简称 |
| 公司名称 | 公司全称 |
| 上市日期 | 上市日期 |
| 发行价格 | IPO 发行价 |
| 行业分类 | 所属行业 |
| 主营业务 | 主营业务描述 |
| 办公地址 | 公司办公地址 |
| 公司网址 | 官网 URL |
| 当前价格 | 最新价 |
| 市净率PB | 市净率 |
| 市盈率TTM | 滚动市盈率 |
| 总市值 | 总市值（亿/万自动换算） |
| 流通市值 | 流通市值（亿/万自动换算） |

分红信息（`-d` 选项，表格输出）：

| 列名 | 说明 |
|------|------|
| 公告日期 | 分红公告日期 |
| 分红送配 | 每 10 股送/转/派详情 |
| 除权除息日 | 除权除息日期 |
| 股权登记日 | 股权登记日期 |
