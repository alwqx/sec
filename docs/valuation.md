# 股票估值方法调研

## 概述

股票估值方法分为两大类：**绝对估值法**（计算内在价值）和**相对估值法**（与同类公司比较）。不同行业、不同生命周期阶段的公司适用不同的估值方法。

---

## 一、绝对估值法

### 1.1 自由现金流折现模型 (DCF)

最经典的绝对估值方法，将公司未来产生的自由现金流折现到当前。

**公式：**

```text
企业价值 = Σ (FCF_t / (1 + WACC)^t) + 终值 / (1 + WACC)^n
股权价值 = 企业价值 - 净负债
每股价值 = 股权价值 / 总股本
```

**两种 FCF：**

| 类型                   | 公式                                                 | 适用             |
| ---------------------- | ---------------------------------------------------- | ---------------- |
| FCFF（公司自由现金流） | EBIT×(1-t) + 折旧摊销 - 资本支出 - 营运资本增加      | 计算企业价值     |
| FCFE（股权自由现金流） | 净利润 + 折旧摊销 - 资本支出 - 营运资本增加 + 净借款 | 直接计算股权价值 |

**WACC 计算：**

```text
WACC = E/(E+D) × Ke + D/(E+D) × Kd × (1-t)

Ke = Rf + β × (Rm - Rf)   (CAPM)
```

**所需指标：**

| 指标                 | 来源                      | sec 现有                   |
| -------------------- | ------------------------- | -------------------------- |
| 净利润 (Net Profit)  | 利润表                    | ✓ `sec bs --type income`   |
| EBIT（息税前利润）   | 利润表（可推算）          | ✓                          |
| 折旧与摊销           | 现金流量表                | ✓ `sec bs --type cashflow` |
| 资本支出 (CapEx)     | 现金流量表                | ✓                          |
| 营运资本变动         | 资产负债表                | ✓ `sec bs --type balance`  |
| 无风险利率 Rf        | 国债收益率                | ✓ `sec bond`（美债 10Y）   |
| Beta (β)             | 市场数据计算              | ✗ 需实现                   |
| 市场风险溢价 (Rm-Rf) | 历史数据                  | ✗ 需外部数据               |
| 总股本               | 公司信息                  | ✓ `sec info`               |
| 净负债               | 资产负债表                | ✓                          |
| 税率                 | 利润表（所得税/利润总额） | ✓                          |
| 永续增长率 g         | 主观估计                  | ✗ 需假设                   |

**优缺点：** 理论上最严谨，但对假设极度敏感（WACC 和 g 的微小变动导致结果大幅波动），适合现金流稳定的成熟公司。

---

### 1.2 股利折现模型 (DDM)

假设股票价值等于未来所有股利的现值之和。

**公式：**

```text
戈登增长模型（恒定增长）：
P = D₁ / (r - g)

多阶段模型：
P = Σ (D_t / (1+r)^t) + D_n×(1+g₂) / ((r-g₂)×(1+r)^n)
```

**所需指标：**

| 指标           | 来源              | sec 现有                 |
| -------------- | ----------------- | ------------------------ |
| 每股股利 (DPS) | 分红公告 / 利润表 | ✓ `sec info --dividends` |
| 股利增长率 g   | 历史数据估算      | 可从分红历史推算         |
| 要求回报率 r   | CAPM 或主观       | 同 DCF                   |
| 派息率         | DPS/EPS           | ✓ 可计算                 |

**适用：** 分红稳定、派息率高的成熟公司（银行、公用事业）。

---

### 1.3 剩余收益模型 (Residual Income)

将内在价值分解为当前账面价值 + 未来超额收益现值。

**公式：**

```text
P = BVPS₀ + Σ ((ROE_t - r) × BVPS_{t-1}) / (1+r)^t
```

**所需指标：**

| 指标                | 来源                | sec 现有 |
| ------------------- | ------------------- | -------- |
| 每股净资产 (BVPS)   | 资产负债表 / 总股本 | ✓        |
| ROE（净资产收益率） | 利润表 / 净资产     | ✓        |
| 要求回报率 r        | CAPM                | 同上     |
| 总股本              | 公司信息            | ✓        |

**优势：** 对终值假设的敏感性低于 DCF，适合净资产较大的公司（银行、保险）。

---

## 二、相对估值法（乘数法）

### 2.1 市盈率 (P/E)

最常用的估值乘数。

**公式：**

```text
合理股价 = 合理市盈率 × 每股收益 (EPS)

P/E = 股价 / EPS
```

**三种 P/E：**

| 类型               | EPS 来源     | 含义         |
| ------------------ | ------------ | ------------ |
| 静态 P/E (TTM)     | 最近四个季度 | 已实现的盈利 |
| 动态 P/E (Forward) | 分析师预测   | 预期盈利     |
| 滚动 P/E           | 最近 12 个月 | 平滑季节性   |

**所需指标：**

| 指标             | 来源     | sec 现有                 |
| ---------------- | -------- | ------------------------ |
| 当前股价         | 行情数据 | ✓ `sec quote`            |
| EPS（基本/稀释） | 利润表   | ✓ `sec bs --type income` |
| 行业平均 P/E     | 外部数据 | ✗ 需外部                 |
| 历史 P/E 区间    | 历史数据 | ✓ 可从历史行情+财报推算  |
| 盈利增长率       | 历史对比 | ✓ 同上年比               |

**适用：** 盈利稳定且为正的公司。不适用于亏损公司、周期性行业顶部。

---

### 2.2 市净率 (P/B)

适用于重资产公司。

**公式：**

```text
合理股价 = 合理市净率 × 每股净资产 (BVPS)

P/B = 股价 / BVPS = 总市值 / 净资产
```

**所需指标：**

| 指标              | 来源              | sec 现有 |
| ----------------- | ----------------- | -------- |
| 当前股价          | 行情              | ✓        |
| 每股净资产 (BVPS) | 资产负债表        | ✓        |
| ROE               | 利润表/资产负债表 | ✓        |
| 行业平均 P/B      | 外部数据          | ✗        |

**P/B 与 ROE 的关系：** 高 ROE 的公司理应享有高 P/B。理想情况下 `P/B ≈ ROE / r`。

**适用：** 银行、保险、地产等重资产行业；不适用于轻资产/技术公司。

---

### 2.3 市销率 (P/S)

适用于成长型或暂时亏损的公司。

**公式：**

```text
P/S = 总市值 / 营业收入
     = 股价 / 每股营收
```

**所需指标：**

| 指标         | 来源        | sec 现有     |
| ------------ | ----------- | ------------ |
| 总市值       | 行情×总股本 | ✓ `sec info` |
| 营业收入     | 利润表      | ✓            |
| 行业平均 P/S | 外部数据    | ✗            |

**适用：** 营收稳定但尚未盈利的成长公司；周期性底部公司。

---

### 2.4 企业价值倍数 (EV/EBITDA)

比 P/E 更全面的估值乘数，考虑了资本结构和折旧政策差异。

**公式：**

```text
EV = 总市值 + 净负债 + 少数股东权益

EBITDA = EBIT + 折旧 + 摊销

合理 EV = 行业 EV/EBITDA × 公司 EBITDA
```

**所需指标：**

| 指标               | 来源       | sec 现有 |
| ------------------ | ---------- | -------- |
| 总市值             | 行情       | ✓        |
| 净负债             | 资产负债表 | ✓        |
| EBIT               | 利润表     | ✓        |
| 折旧摊销           | 现金流量表 | ✓        |
| 行业平均 EV/EBITDA | 外部       | ✗        |

**适用：** 重资产行业、并购估值、跨市场比较。

---

### 2.5 PEG 指标

将 P/E 与盈利增长率结合，衡量估值是否合理。

**公式：**

```text
PEG = P/E / 盈利增长率 (%)

合理 PEG = 1（即 P/E 应等于增长率）
```

**所需指标：**

| 指标                      | 来源        | sec 现有 |
| ------------------------- | ----------- | -------- |
| TTM P/E                   | 行情+利润表 | ✓        |
| 净利润增长率（近 3-5 年） | 历史利润表  | ✓        |
| 预期增长率                | 分析师预测  | ✗        |

**适用：** 成长型公司。PEG < 1 可能低估，PEG > 2 可能高估。

## 三、其他方法

### 3.1 格雷厄姆公式 (Graham Number)

本杰明·格雷厄姆提出的简单估值方法。

**公式：**

```text
格雷厄姆数 = √(22.5 × EPS × BVPS)

合理买入价 ≤ 格雷厄姆数
```

22.5 来自格雷厄姆的经验值（P/E ≤ 15 × P/B ≤ 1.5）。

**所需指标：** EPS、BVPS（sec 均可提供）。

**适用：** 防御型价值投资，快速筛选低估股票。

---

### 3.2 Peter Lynch 合理 PEGY

将 PEG 与股息率结合。

**公式：**

```text
PEGY = P/E / （盈利增长率 + 股息率）

合理 PEGY < 1
```

**所需指标：** P/E、盈利增长率、股息率（sec 均可提供）。

---

### 3.3 席勒 CAPE （周期调整市盈率）

用 10 年平均通胀调整后的 EPS 替代当前 EPS，平滑经济周期影响。

**公式：**

```text
CAPE = 股价 / 10 年平均通胀调整 EPS
```

**所需指标：** 10 年 EPS 历史（需长期数据）、CPI 数据（外部）。

---

## 四、各方法适用场景汇总

| 行业                | 首选方法                  | 辅助方法  |
| ------------------- | ------------------------- | --------- |
| 银行/保险           | P/B + ROE、DDM、剩余收益  | P/E       |
| 消费/医药（成熟）   | DCF、P/E                  | PEG、P/S  |
| 科技/互联网（成长） | P/S、PEG                  | EV/EBITDA |
| 地产/基建           | P/B、NAV                  | DCF       |
| 周期性行业          | P/B（底部）、P/E 均值回复 | EV/EBITDA |
| 亏损公司            | P/S、P/B                  | EV/EBITDA |
| 公用事业            | DDM、DCF                  | P/E       |

## 六、实现建议

### 6.1 可立即实现的功能

基于 sec 现有数据，可以立刻实现以下估值指标的计算：

```
sec valuation 600036 --method pe          # P/E + 历史 PE 带
sec valuation 600036 --method pb          # P/B + ROE
sec valuation 600036 --method ps          # P/S
sec valuation 600036 --method peg         # PEG = PE / 增长率
sec valuation 600036 --method graham      # 格雷厄姆数
sec valuation 600036 --method overview    # 多方法综合估值摘要
```

### 6.2 需要额外数据的功能

| 功能     | 缺什么                       | 方案                                         |
| -------- | ---------------------------- | -------------------------------------------- |
| DCF 估值 | Beta、市场风险溢价、增长预测 | Beta 可从行情回归计算；其他用默认值+可调参数 |
| 行业比较 | 行业平均 P/E、P/B            | 可接入东方财富行业数据 API                   |
| CAPE     | 10 年 EPS + CPI              | 可计算但需 10 年数据积累                     |

### 6.3 DCF 的默认参数策略

DCF 的关键假设可以让用户指定，也可以用合理默认值：

```bash
sec valuation 600036 --method dcf \
    --growth-rate 5%        \   # 未来 5 年增长率（默认：近 3 年均值）
    --terminal-growth 3%    \   # 永续增长率（默认：GDP 增速）
    --wacc 8%               \   # 折现率（默认：CAPM 自动计算）
    --margin-of-safety 20%      # 安全边际
```
