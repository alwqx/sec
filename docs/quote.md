# 证券行情数据

## sec quote (`sec q`)

查询证券实时行情，数据来源于[新浪财经](https://finance.sina.com.cn)。

```
sec quote <codes>
sec q <codes>
```

### 参数

| 参数 | 说明 |
|------|------|
| `<codes>` | 证券代码，多个用逗号分隔，最多 5 个。支持 A 股（`sh601222`）、港股（`hk09988`）、美股（`msft`） |

### 选项

| 选项 | 说明 |
|------|------|
| `-r, --realtime` | 实时模式，每 3 秒刷新一次（Ctrl-C 退出） |
| `-D, --debug` | 开启 debug 日志 |

### 实现架构

```
cmd/quote/quote.go        -- CLI 命令入口
provider/sina/sina.go     -- 证券搜索
provider/sina/quote.go    -- 行情查询（HTTP + WebSocket）
```

### 数据流程

1. `sina.MultiSearch(ctx, keys)` — 并发搜索证券代码，获取 `ExCode` 等信息
2. `sina.QueryQuoteList(ctx, codes)` — 批量 HTTP 查询实时行情
3. `printQuote` — 使用 `tablewriter.Rich()` 渲染表格

### 输出字段

| 列名 | 说明 |
|------|------|
| 时间 | 交易日期 + 时间 |
| 名称 | 证券名称 |
| 当前价格 | 最新价 + 涨跌额 + 涨跌幅（红涨绿跌） |
| 昨收 | 昨日收盘价 |
| 今开 | 今日开盘价 |
| 最高 | 当日最高价 |
| 最低 | 当日最低价 |
| 成交量 | 成交量（手），大数以 万/亿 显示 |
| 成交额 | 成交额（元），大数以 万/亿 显示 |
| 证券代码 | 交易所代码 |

### 输出示例

```bash
$ sec q sh688047,s
时间                | 名称    | 当前价格          | 昨收   | 今开   | 最高    | 最低   | 成交量  | 成交额    | 证券代码
2024-09-30 15:00:01 | 龙芯中科 | 119.62 19.94 20% | 99.68 | 106   | 119.62 | 104.5 | 825.67万 | 9.38亿   | SH688047
```

## sec quote-history (`sec qh`)

查询证券历史 K 线数据，数据来源于[东方财富](https://www.eastmoney.com/)。

```
sec quote-history <code>
sec qh <code>
```

### 参数

| 参数 | 说明 |
|------|------|
| `<code>` | 证券代码，如 `sh601222`、`hk09988` |

### 选项

| 选项 | 说明 |
|------|------|
| `-b, --begin` | 起始日期，格式 `20250101`，默认最近 30 天 |
| `-e, --end` | 结束日期，格式 `20250131`，默认今天 |
| `-d, --desc` | 按日期倒序排列 |
| `-f, --fq` | 复权类型：`bfq`（不复权，默认）/ `qfq`（前复权）/ `hfq`（后复权） |
| `-D, --debug` | 开启 debug 日志 |

### 实现架构

```
cmd/quote/quote_history.go     -- CLI 命令入口
provider/sina/sina.go          -- 证券搜索
provider/eastmoney/eastmoney.go -- K线历史数据
```

### 数据流程

1. `sina.Search(ctx, key)` — 搜索证券代码
2. 根据交易所确定 `MarketCode`：sh→1, sz→0, hk→116, nasdaq→105
3. `eastmoney.GetQuoteHistory(ctx, req)` — 查询历史 K 线数据
4. `printQuoteHistory` — 使用 `tablewriter.Rich()` 渲染表格

### 输出字段

| 列名 | 说明 |
|------|------|
| 日期 | 交易日 |
| 名称 | 证券名称 |
| 收盘 | 收盘价 + 涨跌额 + 涨跌幅（红涨绿跌） |
| 开盘 | 开盘价 |
| 最高 | 最高价 |
| 最低 | 最低价 |
| 成交额 | 成交额，大数以 万/亿 显示 |
| 成交量 | 成交量，大数以 万/亿 显示 |
| 振幅 | 当日振幅 |
| 换手率 | 换手率 |
| 证券代码 | 市场代码 + 证券代码 |

2024-09-02 | 龙芯中科 | 99.68  5.68  6.1% | 95.00  | 100.50 | 94.50 | 3.2亿  | 320万  | 6.4    | 2.1   | SH688047
