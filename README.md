## 安装

进入 [releases](https://github.com/alwqx/sec/releases) 页面，下载指定操作系统的二进制文件。

Mac/Linux 要把二进制文件放在 `PATH` 路径下。

Windows 要把二进制文件放在 `系统环境变量` 下。
